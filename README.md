# Astro-Mix-Blog
- Inputs(HTML, Markdown, PDF, Images, etc.)
- Outputs(Index Page + About Page)

## Features
- Optional Date Prefix 
    - `[YYYY-MM-DD]filename.extname'
    - `filename.extname'` (Date will be determined by the file updated date)
- Pin
- Pagination
- RSS
- SEO
- Configurable
<!-- - Search -->

## Dependencies
- Node.js
- cheerio
- Astro
- pnpm (Optional)
- Quarto (Optional)

## Get Started
```bash
pnpm i

pnpm build

pnpm pre # prebuild and preview

pnpm dev

pnpm clean # remove all generated files
```
